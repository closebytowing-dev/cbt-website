## üö® COMMON ISSUES & FIXES

### "Analytics not showing data"
- Wait 24-48 hours for first data
- Check IDs are correct in layout.tsx
- Use Google Tag Assistant extension
- View Realtime reports (not regular)

### "Search Console verification fails"
- Ensure code is exact (no extra spaces)
- Redeploy website after adding code
- Wait 5 minutes, try again
- Use DNS TXT record if HTML tag fails

### "Sitemap not indexing"
- Check accessible: visit sitemap.xml
- Validate at xml-sitemaps.com
- Resubmit in Search Console
- Request indexing for individual URLs
